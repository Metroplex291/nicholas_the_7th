About

This is a discord bot that I made for my friends and I to use in our discord server. It is a work in progress and I will be adding more features as I go along.
It is not meant to be used by anyone else, but if you want to use it, feel free to do so.

Installation

To install the bot, you will need to have python 3.8 or higher installed on your computer. You will also need to have pip installed.
To install the bot, follow the instructions on the github page.

Commands

The bot has a few commands that you can use.

General

/help - Shows this message
/ping - This command returns the latency
/language - Change the language for this server
/sound_ effects - Shows all available sound effects
/list_radios - Shows all available radios
/key - Shows the key for the server

Player

/play - Plays the songs :)
/radio - Plays radio from list
/ps - Plays sound effects by number
/skip - Skip song in queue
/nowplaying - Shows what is playing right now
/last - Shows the last song played
/loop - Turn on loop mode
/loop_this - This command loops currently playing song

Queue

/queue - Add song to queue
/remove - Removes song from queue
/clear - Clears the queue
/shuffle - Shuffles the queue
/show - Shows the queue
/search - Search youtube for the provided query

Voice

/stop - Stop the player
/pause - Pause the player
/resume - Resume the player
/join - Bot joins voice channel
/disconnect - Bot leaves voice channel
/volume - Set the volume of the player

Context Menu

Add to queue - Add song to queue
Show Profile - Shows all available information about the profile
